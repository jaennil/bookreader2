# Полка

Адаптивный прототип браузерной читалки для PDF, EPUB и FB2.

## Запуск

```bash
npm install
npm run dev
```

Сборка для публикации:

```bash
npm run build
```

## Что реализовано

- библиотека, поиск, фильтры и избранное;
- загрузка PDF, EPUB и FB2;
- чтение EPUB с пагинацией, FB2 как адаптивного текста и PDF во встроенном просмотрщике;
- сохранение файлов в IndexedDB, а прогресса и настроек — в localStorage;
- профиль пользователя, светлая, тёплая и тёмная темы чтения;
- адаптивный интерфейс с нижней навигацией на мобильных устройствах.

Прототип не содержит серверной части. Для настоящего аккаунта и синхронизации между устройствами нужны API, авторизация и файловое хранилище.

Для production-версии данные профиля и позиции чтения стоит хранить в PostgreSQL, сами книги — в S3-совместимом хранилище, а клиенту отправлять изменения прогресса небольшими фоновыми запросами. Это позволит продолжать чтение на телефоне с той же страницы, где пользователь остановился на компьютере.
