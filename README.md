# Полка

Браузерная читалка с React + TypeScript клиентом и Go API. Поддерживает PDF, EPUB и FB2, аккаунты и синхронизацию прогресса между устройствами.

## Запуск

Требования: Node.js 22+ и Go 1.26+.

```bash
npm install
npm run dev
```

Откройте `http://localhost:5173`. Команда одновременно запускает:

- Vite на порту `5173`;
- Go API на порту `8080`;
- прокси `/api` с Vite на Go.

Если порт занят, Vite напечатает новый адрес в терминале.

## Проверка

```bash
npm test
```

Команда запускает интеграционные тесты Go API, проверку TypeScript и production-сборку клиента.

## Production

```bash
npm run build
go run ./backend/cmd/server
```

Go-сервер раздаёт собранный клиент из `dist` и API на одном адресе `http://localhost:8080`.

Переменные окружения:

- `PORT` — порт сервера, по умолчанию `8080`;
- `DATA_DIR` — каталог состояния и книг, по умолчанию `data`;
- `WEB_DIR` — каталог React-сборки, по умолчанию `dist`.

## Хранение данных

- аккаунты, сессии, метаданные и прогресс: `data/state.json`;
- загруженные книги: `data/books/`;
- пароли: PBKDF2-HMAC-SHA256 с индивидуальной солью;
- сессии: случайные Bearer-токены со сроком жизни 30 дней.

Каталог `data` исключён из Git. Для публичного production-развёртывания нужны HTTPS, резервное копирование каталога данных и внешний reverse proxy.
